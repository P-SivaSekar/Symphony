# Symphony

A modern, elegant, and feature-rich music player application built with Flutter.

## 🎵 Features
* **Beautiful UI:** Enjoy a clean and intuitive interface for browsing and playing music.
* **Seamless Playback:** Reliable audio and video playback using modern libraries.
* **Cloud Sync:** Firebase integration for saving your favorites and data in the cloud.
* **Cross-Platform:** Available as a native Android app and an accessible Web app.

## 🚀 Getting Started

### 📱 Android Installation

1. Navigate to the **Releases** section of this repository.
2. Download the latest `Symphony.apk` file (e.g., Version 1.0.0).
3. Transfer the APK file to your Android device (if downloaded on a PC).
4. On your device, open a File Manager and tap the `Symphony.apk` to install.
   > **Note:** You may need to enable "Install from Unknown Sources" or "Install unknown apps" in your Android settings.
5. Launch Symphony and enjoy your music!

### 🌐 Web Access

You can also access Symphony directly from your browser without any installation! 
Visit the web app here: [Symphony Web App](https://symphony-music-app-6eddc.web.app/)
